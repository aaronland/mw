Abstract:

This mini-workshop will give an overview of search engine technology currently available. Themes covered will be metadata, indexing strategies, what can be indexed, what can't etc. Understanding how a search engine works will assist those interested in choosing a search engine for their own web site. A comparison of software will then give an insight of how much mileage the user can get out of freeware and commercial products. The session will finish with a "hands-on" example showing how easy it can be to implement a search engine for a web site.
