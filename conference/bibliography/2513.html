Title:The Museum Top-Level Domain: An Update
Authors:Cary Karp
Type:Paper
Publication:MW2002: Museums and the Web 2002
Year:2002
Abstract:

Last year, the application for a .museum top-level domain name submitted by MuseDoma (http://musedoma.museum/)was accepted by ICANN. Since then, detailed planning and implementation has been underway and the domain is currently fully operational. Cary Karp will report on the status and plans.
