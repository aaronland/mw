Title:User Evaluation in the Retrieval of Digital Art Images
Authors:Samantha Hastings, Theresa Russell
Type:Paper
Publication:MW99: Museums and the Web 1999
Year:1999
Abstract:

Intellectual access to art images is the area of concentration for this study in the analysis of user supplied index terms. The research project investigates user evaluation in a broader context for retrieval of images on the Web. Methods include online user surveys, log analysis, collection of user supplied access terms and correlation of user queries to retrieved sets of images. Preliminary results support the need for improved browsing tools, image manipulation software, feedback mechanisms, query analysis and the role of user evaluation in retrieval on the Web.
