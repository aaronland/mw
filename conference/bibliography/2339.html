Title:The Use of FileMaker for Museum Databases (Collection, Contact, and Program Databases)
Authors:James Swanson
Type:Paper
Publication:MW99: Museums and the Web 1999
Year:1999
Abstract:

Databases can be configured in a number of ways to create web pages. Using Filemaker as an example, this workshop will discuss procedures available to create html documents from collection and programming databases. Methods of exporting fields to create files that can be uploaded to internet hosts will be demonstrated. The use of Filemaker as a web server to directly create web pages, for museums that host their own site or maintain an intranet, will be briefly discussed.
