Title:Quick and Dirty Dublin Core: Examples with FileMaker
Authors:James Swanson
Type:Paper
Publication:MW2000: Museums and the Web 2000
Year:2000
Abstract:

This workshop will demonstrate how to create Dublin Core-compliant markup from FileMaker database records. Scripting techniques and calculations will be used to convert field values to proper formats, to import values from related files, to create multiple elements from fields with multiple values, and to convert character entities. A FileMaker extension will be used to create and name exported XML files. The techniques will be applicable to creating other types of XML markup, such as RDF (resource description framework) and RSS (rich site summary) files. The techniques would also be applicable to other scriptable database applications.
